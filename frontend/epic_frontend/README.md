## API Endpoints
                    
Endpoint  | Contents
------------- | -------------
`/`  |  Connection test
`/books/:bookId`  | GET: Shows an individual book
`/books/:bookId/return`  | PUT: Update the system for a book that has been returned
`/books/:bookId/chekout/:userID`  | GET/POST: Check out a book based on a selection
